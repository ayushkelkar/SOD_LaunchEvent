# SOD Club — Round 3 Treasure Hunt Frontend
**Silicon Multiverse · BMSIT & M**

## Setup
```bash
npm install
npm run dev
# Runs on http://localhost:3001
```

## Build for Render
- Build Command: `npm install && npm run build`
- Publish Directory: `dist`

## Before deploying
1. Copy `SOD_LOGO` and `BMS_LOGO` base64 constants from Round 1 `App.jsx` into `src/App.jsx`
2. Update `API.BASE_URL` with Round 3 backend URL
